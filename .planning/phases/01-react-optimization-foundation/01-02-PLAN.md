<objective>
Update all Zustand store subscriptions to use narrow selectors with shallow comparison, preventing cascading re-renders when unrelated state changes.

Purpose: Currently components like OtherPlayers use `useGameStore()` without selectors, causing re-renders when ANY slice changes (projectiles added, effects update, etc.). Narrow selectors ensure components only re-render when their specific data changes.

Output: Components using narrow Zustand selectors with shallow comparison, eliminating unnecessary re-renders.
</objective>
