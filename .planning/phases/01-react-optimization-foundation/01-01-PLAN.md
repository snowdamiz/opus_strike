<objective>
Replace all setState calls in useFrame hooks with direct ref mutations to eliminate 60fps re-render cascades.

Purpose: setState in useFrame triggers React re-renders at 60fps, causing cascading re-renders across all effect components. This is the #1 performance anti-pattern in React Three Fiber apps per official R3F documentation.

Output: Effect components using ref mutations for high-frequency updates, eliminating unnecessary re-renders.
</objective>
