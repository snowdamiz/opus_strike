<objective>
Migrate player position interpolation from props-based updates to visualStore-based updates. OtherPlayers will read target positions from visualStore in useFrame (non-reactive), and PlayerController will write local player position to visualStore after each frame update.

Purpose: Eliminate React re-renders caused by player position updates. Player positions update at 60fps during network sync, but only visual interpolation should run at 60fps - not React re-renders.

Output: OtherPlayers uses visualStore for position interpolation, PlayerController updates visualStore, zero React re-renders from 60fps player movement.
</objective>
