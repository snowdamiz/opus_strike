2. Update initialize() method to use configured positions:

```typescript
initialize(): void {
  const { spawnPoints, flagZones } = SCI_FI_CTF_POSITIONS;

  // Set up spawn points from map config
  this.spawnManager.setSpawnPoints('red', spawnPoints.red);
  this.spawnManager.setSpawnPoints('blue', spawnPoints.blue);

  // Initialize CTF with configured flag positions
  this.ctfMode.initialize(
    flagZones.red,
    flagZones.blue
  );
}
```

Note: We call setSpawnPoints() directly with the full arrays rather than initialize(). The SpawnManager.initialize() generates default positions around a base point, but setSpawnPoints() replaces those with our explicit positions.

The old code was:
```typescript
this.spawnManager.initialize(
  { x: -40, y: 1, z: 0 },
  { x: 40, y: 1, z: 0 }
);
this.ctfMode.initialize(
  { x: -40, y: 1, z: 0 },
  { x: 40, y: 1, z: 0 }
);
```

New code uses positions matching actual map geometry:
- Spawn points at x=-80 (red) and x=80 (blue)
- Flag zones at x=-90 (red) and x=92 (blue)
